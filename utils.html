Utilities¶

tomcrypt.utils.pem_decode(content)¶

PEM decode a key.

Returns a tuple of:
type – “RSA”, “EC”, or None (likely if public) mode – “PRIVATE” or “PUBLIC” content – Decoded content.

Throws a tomcrypt.Error if content is not PEM encoded.

tomcrypt.utils.pem_encode(type, mode, content)¶

PEM encode a key.

Params:
type – “RSA”, “EC”, etc. mode – “PUBLIC” or “PRIVATE” content – The thing to encode.
